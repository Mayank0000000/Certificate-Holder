
# Project Title

Certificate Holder

## Run Locally

Clone the project

```bash
  git clone https://github.com/Mayank0000000/Certificate-Holder
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
